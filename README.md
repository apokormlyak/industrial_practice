# industrial_practice

Для установки зависимостей использовать команду:
`pip install -r requirements.txt`

Проект состоит из файла запуска программы main.py, модулей worker.py и utils.py, а также из файла с тестами.

## Программа запускается командой:
`python main.py`

## Выполнение программы идет по следующему флоу:
- запуск программы, вызов функции run()
- пока пользователь не введет опцию 3, выполнение программы продолжается (опция 3 заканчвает выполнение программы)
- опция 1 запускает создание нового сотрудника
- опция 2 запускает вывод сотрудников, работающих в компании более заданного количества лет

### Класс Worker:
- состоит из защищенных атрибутов 
- доступ к атрибутам осуществляется с помощью геттеров, обернутых в декоратор property

### Тесты:
- фреймворк для запуска тестов - pytest
- на данный момент, за неимением иного функционала, покрыты только существующие модули (unit-тесты)

Для запуска тестов использовать команду:
`pytest tests`
